# Adidas-Analysis
📊 Adidas US Sales Analysis (Power BI Dashboard)

📌 Description
This project presents a **Power BI dashboard analyzing Adidas US sales data**. It helps understand **sales performance, product categories, retailers, regional distribution, and profit trends** through interactive visualizations.

---

📂 Dataset / Inputs

The dataset includes:

* Retailer
* Region and State
* Product Category
* Units Sold
* Price per Unit
* Total Sales
* Operating Profit
* Operating Margin
* Sales Date

File format: **.pbix**

---

🛠 Tech Stack

* Power BI
* Excel / CSV (data source)
* Data Modeling

---

⚙️ Installation

* Clone the repository
* Install **Power BI Desktop**
* Open the `.pbix` file to view the dashboard

---

📊 Key Insights

The dashboard helps analyze:

* Total sales and profit
* Best-performing regions
* Top retailers
* Product category performance
* Sales trends over time
